# RedRoster API

API documented [here](http://docs.redroster.apiary.io/)



## Run Locally

To run this locally with the ability to interface with an iOS app, run

```
rails s --binding=0.0.0.0
```

Routes are attached to:

```
<IP_ADDRESS>:<PORT>/ROUTE
```
